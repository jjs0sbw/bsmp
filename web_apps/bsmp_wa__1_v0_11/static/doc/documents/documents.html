Basic Structural Modeling Web Application 1

Basic structural modeling is a component of structural modeling.

The documents associated with this web application are grouped into three
basic types:Instructions, Background, and Other. The Basic Structural Modeling
Project documents are located in the Background section. The documents include the
Problem Description Report, Conceptual Solution Report and the Detailed Design Report.

Operation instruction documents will be developed and placed in the Instructions
section.

Documents will be added as they are developed and the program is improves.
